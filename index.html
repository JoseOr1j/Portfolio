A web card game made using, HTML, CSS, and Vanilla Javascript. The game uses sounds and has a clean user interface. The user is given 100 seconds to match all the cards, with sounds for instances such as, flipping, matching, winning, losing, background music, etc. Using a card shuffling algorithm to ensure that the games are almost never alike.

C++ Sudoku Game

C++, Visual Studio IDE

This console application solves various Sudoku games with speedy succession. There are missing values of a winnable Sudoku game, my program loops around and finds what integers are needed to complete the sudoku game, if there are none then the program moves to the next game, when all are completed the game terminates and thanks the user for playing.


Background: This game was made using C++. I used multi-dimensional arrays, functions, loops, and more to solve a 9x9 sudoku game.
